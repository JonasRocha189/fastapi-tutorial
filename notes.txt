Put // usually updates the entire content
patch // usually updates a part of the content
obs: In practice just uses the patch (partial)

Delete: usually returns 204

Status code 
200 - ok // GET, PUT or PATCH
201 - Created // Successful POST for users and posts
204 - No Content // Successful DELETE
400 - Bad Request // Duplicate username/email when creating user 
404 - Not Found // Resource doesn't exists (user or post)
422 - Unprocessable Entity // Validation error (automatic from Pydantic)

AWS S3 and Boto3

uv add boto3

aws_bucket_policy.json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadProfilePics",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::fastapi-blog-uploads-jonas/profile_pics/*"
    }
  ]
}


aws_iam_policy.json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": ["s3:PutObject", "s3:DeleteObject"],
      "Resource": "arn:aws:s3:::fastapi-blog-uploads-jonas/profile_pics/*"
    }
  ]
}
