# Blog Project

A modern RESTful Blog API built with FastAPI and Pydantic, designed with clean architecture principles and scalable backend development practices.

---

Tools used
- FastAPI
- Pydantic
- uv
- Jinja templates


---

To run the project

` uv pip -r install requirements.txt `

or creating a uv project from scratch, first create a uv project

` uv init ` them install the dependencies ` ud add -r requirements.txt `
